Features

REQUEST URL HTTP METHOD QUERY PARAMETER REQUEST DATA FORMAT RESPONSE DATA BODY FORMAT
Get a list of all movies /movies GET None A JSON object holding data about all the movies
Get data (description, genre, director, image URL) about a single movie by title to the user /movies/:title GET :title None A JSON object hold data about the specified movie
Get data about a genre (description) by name/title /movies/genre/:genreName GET :genreName None A JSON object holding data about movies that are in a specific genre.
Get data about a director (bio, birth, year, death year) by name /movies/directors/:directorName GET :directorName None A JSON object holding data about the specified director.
Add new user /users POST A JSON object holding data about the new user to add. A success message to user on success/error of registration status
Update user info based on ID /account/:id PUT :id A JSON object holding data about the updated info for the specified username A success message to user on success/error of update status
Add a new movie to a user's favorite movie list /account/:id/:movieTitle POST :id, :movieTitle A JSON object holding data for a movie A success message to user on success/error of adding a movie to favorite
Remove a movie from a user's favorite movie list /users/:id/:movieTitle DELETE :id, :movieTitle None A success message to user on success/error of removing a favorite movie
Delete a user based on specified user id /users/:id DELETE :id None A success message to user on success/error of deregistering the user
