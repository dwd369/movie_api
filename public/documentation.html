Add new user /users POST A JSON object holding data about the new user to add in the following format:
{
Username: "testuser123",
Password: "123456",
Email: "testuser@test.com",
FavoriteMovies: [ObjectId(MovieObjectId)]
}
 If success, a json file containing the user data will be return in the following format. If error, an error message will be sent back.
{
Username: "testuser123",
Password: "123456",
Email: "testuser@test.com",
FavoriteMovies: [ObjectId(MovieObjectId)]
}
Update user info based on username /users/:Username PUT :Username A JSON object holding data about the updated info for a specified username, the request body should be in the following format:
{
Username: "testuser123",
Password: "123456",
Email: "testuser@test.com",
FavoriteMovies: [ObjectId(MovieObjectId)]
}
 A success message to user on success/error of update status
Add a new movie to a user's favorite movie list /users/:Username/:MovieID POST :Username, :MovieID None If success, a json file containing the user data will be return in the following format. If error, an error message will be sent back.
{
Username: "testuser123",
Password: "123456",
Email: "testuser@test.com",
FavoriteMovies: [ObjectId(MovieObjectId)]
}
Remove a movie from a user's favorite movie list /users/:Username/:movieID DELETE :Username, :movieID None If success, a json file containing the user data will be return in the following format. If error, an error message will be sent back.
{
Username: "testuser123",
Password: "123456",
Email: "testuser@test.com",
FavoriteMovies: [ObjectId(MovieObjectId)]
}
Delete a user based on specified user id /users/:Username DELETE :id None A success message to user on success/error of deregistering the user
